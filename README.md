# PhonePe-Data-Analysis



This project is an end-to-end data analytics solution where I extracted PhonePe transaction data from JSON files, transformed it into structured datasets, and stored it in a SQL database. I performed data analysis using SQL queries to identify transaction trends, user behavior, and insurance insights. Finally, I developed an interactive Streamlit dashboard that allows users to explore insights across states, districts, and time periods. The project helps in understanding digital payment patterns and supports data-driven business decisions.
